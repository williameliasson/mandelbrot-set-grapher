# Mandelbrot set grapher

This is a simple CLI application that shows the mandelbrot set in the console using ASCII characters in a grid.
## How to compile
To compile the program and run the program, execute the following command:
```
gcc mandelbrot.c -o bin/out; ./bin/out
```
## How to use
Use characters N/E/W/S to go a certain direction (N for North, etc) and + or - to zoom in and out.
The Mandelbrot set will be displayed for that area.
